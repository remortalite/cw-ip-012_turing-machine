# Интерпретатор языка программирования Машины Тьюринга

_NB: Лента (tape) в программе выглядит как:_
```
c <-> b <-> a <-> [] -> NULL
```
_(В конце списка есть пустой элемент с указателем на NULL)._

Сборка:

```
make all
```

Запуск:

```
./bin/turing
```

Автоформатирование кода:
```
make format
```

Удаление obj-файлов:
```
make clean
```

Запуск тестов:
```
make test && ./bin/turing-test
```
