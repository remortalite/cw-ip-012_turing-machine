# Интерпретатор языка программирования Машины Тьюринга

Пересборка и запуск:

```
make clean && make all && ./bin/turing
```

Автоформатирование кода:
```
make format
```

Удаление obj-файлов:
```
make clean
```

Запуск тестов:
```
make clean && make test && ./bin/turing-test
```

Для дебага использовать:
```
make debug
```
и
```
make debug test
```
